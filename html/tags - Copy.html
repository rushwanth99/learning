HTML Tags Reference

< body
Tag Description
<html> The root element of the HTML document.
<head> Contains metadata for the document (title, link, etc.).
<title> Sets the title of the page (appears in the browser tab).
<body> Contains the visible content of the document.
<h1>, <h2>, <h3>, <h4>, <h5>, <h6> Defines headings of different levels (h1 is the most important, h6 the least).
<p> Defines a paragraph of text.
<br> Inserts a line break.
<hr> Inserts a horizontal rule (divider).
<b> Defines bold text.
<i> Defines italic text.
<u> Defines underlined text.
<strong> Defines strong emphasis (usually bold).
<mark> Defines highlighted text.
<small> Defines smaller text.
<ins> Defines text that is inserted or underlined.
<div> Defines a block-level generic container.
<header> Defines the header of a page or section.
<a> Defines a hyperlink.
<img> Defines an image.
<iframe> Defines an inline frame (embeds another HTML page).
<form> Defines an HTML form.
<input> Defines an input field for various types (text, checkbox, radio, etc.).
<button> Defines a clickable button.
<option> Defines an option inside a <select> dropdown.
<table> Defines a table.
